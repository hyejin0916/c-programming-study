# C Fundamentals
C 언어 기초 학습 내용을 정리한 저장소입니다.

## 학습 내용
- 변수와 자료형
- 연산자
- 조건문
- 반복문
- 함수
- 배열
- 포인터
- 구조체
- 동적 메모리 할당

## 개발 환경
- macOS
- clang++
- C17 / C++17


## 실행 방법
# C++17 표준으로 [file name].cpp를 컴파일하고
# 결과 실행 파일 이름을 test로 지정
clang++ -std=c++17 [file name].cpp -o test

# 현재 디렉터리의 test 프로그램 실행
./test
